# PC Games Recommendation Using Cosine Similarity

The dataset consists of the name of the games and their descriptions. I recommendation system has been made using cosine similarity to recommend new games to users based the how user describes the game.

[Dataset Link](https://www.kaggle.com/datasets/alyahmedts13/top-pc-games-metacritic-vs-steam-popularity/data)
[Kaggle Link](https://www.kaggle.com/code/singhtanveer38/pc-games-recommendation-cosine-similarity)
